# springboot-admin

springboot-admin是基于`springboot2.0.1` 前端采用`vue-element-admin`组件的快速开发脚手架单体服务。

##1 项目技术栈

* 数据库`mysql6.5`

*  spring`springboot2.0.1`

* 登录`jwt token`

*  持久化`tk 通用mapper`

*  接口管理`swagger-ui`

*  数据库连接池`druid`

*  序列化` fastjson`

*  分页 ` pagehelper `

*  自我检测 `actuator`

*  单元测试` junit`

*  数据库`redis`
